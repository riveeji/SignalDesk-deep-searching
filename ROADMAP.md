# 路线图

这份路线图只记录当前项目明确要做、且和产品定位一致的方向。

## P0

- 提升多候选问题的检索覆盖稳定性
- 继续减少“答非所问”与证据失衡时的错误推荐
- 优化首页、详情页和报告页的信息层级
- 提高 README、案例和演示材料的完成度

## P1

- 支持更多可选搜索 provider
- 增强学术来源接入能力
- 改进报告页图表与可视化导出
- 增加来源筛选、排序和分组能力

## P2

- 用户级长期研究记忆
- 可复用研究模板
- 更细粒度的研究轨迹回放
- 在线部署与分享能力

## 明确暂不做

- 通用聊天助手
- 开放式多 agent 平台
- 企业级多租户权限系统
- 自动抓取 `Google Scholar / CNKI`

这些方向不是不重要，而是会让项目偏离“技术选型 Deep Research System”的核心定位。
